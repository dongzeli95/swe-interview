# Interview Tracks

Ben's parallel prep for three role types. Shared drills live at the top of the repo (`algorithm/`, `system-design/`, `behavioral/`); track-specific study plans, curated problem lists, and reading lists live here.

| Track | Target roles | Focus |
| --- | --- | --- |
| [SWE Senior/Staff](./swe-senior-staff/) | FAANG-tier senior/staff SWE | DSA breadth + system design + staff-level behavioral |
| [Quant Trading](./quant/) | Citadel, Two Sigma, Jump Trading, HRT, Optiver | Probability, C++ fluency, brainteasers, low-latency systems |
| [GenAI MLE](./genai-mle/) | AI labs (OpenAI/Anthropic/DeepMind), GenAI teams | LLM engineering, ML system design, GenAI-specific interview loops |

## How to use

1. Pick a track. Read its `README.md` and `roadmap.md`.
2. Study drills come from the shared `algorithm/` and `system-design/` folders — each track's `problems.md` (or `resources.md`) points into those.
3. Track-only material (probability puzzles, ML sys design, GenAI reading list) lives under `tracks/<track>/`.
4. All coding is Python-first (per repo convention). Legacy C++ solutions stay under `algorithm/<category>/cpp/` as reference.
